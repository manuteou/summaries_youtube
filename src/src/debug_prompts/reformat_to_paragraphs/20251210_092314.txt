### Analyse de la Qualité des Leads et Objectifs des Stages

L'évaluation des leads est primordiale.  Il est crucial de se concentrer sur la qualité des contacts, plutôt que sur le volume. L'objectif est d'appeler tous les étudiants, afin de maximiser le nombre d'intégrations au CRM.  L'équipe a constaté que les applications pour les stages de seconde sont souvent faites à la dernière minute. Pour minimiser l'effort, l'équipe utilise une automatisation pour traiter les nouvelles données qui arrivent dans le fichier, en copiant et collant automatiquement les informations dans la source de données.  Pour le suivi des leads, l'équipe utilise également un fichier Excel pour enregistrer les étapes de contact avec chaque étudiant.  Il est essentiel de suivre les stages réalisés avec les étudiants en seconde. Le but est de voir le nombre de personnes qui ont des rendez-vous avec la web academy. Les données sont donc partagées avec la Web Academy afin de lancer de nouvelles campagnes.
